HTTP

HyperText Transfer Protocol

Limbajul prin care comunica clientul cu un server

HTTP Request methods

  • get - retrieve methods
  • post -send entity to resource
  • put - replace resource
  • head - similar to get, but without body: check if resource exists
  • options - describe comm options for the target resoruce
  • patch - apply partially modifications to a resource
  • delete - delete resource

http response codes

  • 2xx - success
  • 3xx - redirect
  • 4xx - client error
  • 5xx -server error
